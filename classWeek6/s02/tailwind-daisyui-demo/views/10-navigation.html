Navbars

In DaisyUI, the navigation menu is composed of these elements:

  1. navbar container (class = navbar)
  2. navbar items (logo, menu items, etc): Containers for the content to display in the navbar

The position of the navbar items are controlled by these classes

  1. navbar-start: Moves content to start of the navbar
  2. navbar-center: Moves content to center of the navbar
  3. navbar-end: Moves content to end of the navbar

Sometimes, the documentation examples will use the flex properties instead. Both approaches are valid.
