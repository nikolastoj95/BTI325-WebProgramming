Hero Section

Typically, a hero section occupies the full height and width of the screen.

In DaisyUI, the hero is composed of 3 container elements:

  1. hero container (div class = hero)
  2. content container (div class = hero-content)
  3. content wrapper that sets the width of the content (div class = max-w-lg)

The content you want displayed in the hero are placed in the content wrapper element (div class = max-w-lg)

The size of the hero is controlled with these classes:

  1. min-h-screen: sets the hero to the full height/width of the screen
  2. max-w-lg: sets the width of the hero content
